# url의 구성을 알아보자

```
1) protocol
2) host
3) port
4) resource path
5) query
```
```
ex) 1. https:// + 2. www.test.com + 3. :443 + 4. /path/to/resource 5. ?a=b&x=y
```
|protocol|host|port|resource path|query|
|---|---|---|---|---|
|https://|www.test.com|:443|/path/to/resource|?a=b&x=y|
|protocol|host|port|resource path|query|
