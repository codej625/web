# sub domain이란?
```
서브 도메인(subdomain)은 하위 도메인으로 웹사이트의 섹션을 구분하기 위해 도메인 이름에 추가되는 프리픽스(Prefix)를 말한다.
```

```
서브 도메인은 해당 메인 도메인과는 별도의 웹사이트로 작동한다. 
이렇게 구분함으로써 웹사이트의 전반적인 목적성을 유지하면서 웹사이트 다양한 섹션을 개발할 수 있다. 
결과적으로 특정 분야에서 전문성을 구축하여 더 많은 유기적 트래픽을 얻을 가능성이 높아진다.
```

```
https://support.wix.com에서 "support" 부분이 서브 도메인이다.
```

## 서브 디렉토리 (subdirectory) vs 서브 도메인
```
서브 디렉토리 (subdirectory)는 서브 폴더(subfolder)라고 이해하면 쉽다.
검색 엔진은 하위 도메인을 해당 메인 도메인과 별개로 인식하는 반면 서브 디렉토리는 도메인 내부에 중첩된 조직 폴더로 인식한다.
서브 도메인 대신 서브 디렉토리 아래에 콘텐츠를 분리해 놓으면 메인 도메인을 지원하고 있는 것으로 검색 엔진에 표시 된다. 
따라서, 웹사이트의 전체를 보완할 목적으로 더 많은 콘텐츠를 구성하려 한다면 서브 도메인보다 서브 디렉토리를 활용하는 것이 그 목적에 부합할 수 있다.
```
